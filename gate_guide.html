Gate Guide

NOT gate - Also known as the X gate, flips the state of a qubit. If the qubit is in state 0, it changes to state 1, and vice versa. It is essential for creating basic quantum logic operations and is equivalent to the classical NOT gate.

Example:

X|0⟩ → |1⟩
X|1⟩ → |0⟩

Hadamard Gate - The Hadamard gate puts a qubit into an equal superposition of 0 and 1. This means that after applying the Hadamard gate, the qubit has a 50% chance of being measured as 0 or 1. It is crucial for creating quantum parallelism and entanglement.

Example:

H|0⟩ → (|0⟩ + |1⟩)/√2
H|1⟩ → (|0⟩ - |1⟩)/√2

Swap Gate - The Swap gate exchanges the states of two qubits. It is used to reorder qubits within a quantum circuit, which can be important for certain quantum algorithms and operations.

Example:

Swap(|q1⟩, |q2⟩) → (|q2⟩, |q1⟩)

If |q1⟩ = |0⟩ and |q2⟩ = |1⟩, then after the swap:

Swap(|0⟩, |1⟩) → (|1⟩, |0⟩)

CNOT Gate - The CNOT (Controlled-NOT) gate flips the state of a target qubit if the control qubit is in the state 1. It is a fundamental gate for creating entanglement between qubits, which is a key resource in quantum computing.

Example:

CNOT|00⟩ → |00⟩
CNOT|01⟩ → |01⟩
CNOT|10⟩ → |11⟩
CNOT|11⟩ → |10⟩

Noise - Noise is not a gate, but it represents the random disturbances that can affect qubits in a quantum system. Modeling noise is important for understanding and improving the reliability of quantum computations.
